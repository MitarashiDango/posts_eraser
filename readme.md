# Posts eraser

## これはなに？
- 投稿を削除するときにやっつけで作成したスクリプト
- 自分で使うために作ったものなので原則ノンサポートで

## どうやって動かすの
1. ``.env.example``をコピーして``.env``を作成する
```
cp .env.example .env
```

2. 項番1で生成した``.env``ファイルを開き、``ACCESS_TOKEN``へアクセストークンを、``DOMAIN``へ操作対象サーバーのドメインを設定する。

3. 必要に応じてその他の項目を設定する。

4. 以下のコマンドを実行する。
```
npm install
npm run build
```

5. 以下のコマンドを実行すると削除処理が動き始める。
```
npm start
```
